#################### ResortSHOP ####################

El proyecto se basa en una tienda de ropa online.


Para poder hacer funcionar el proyecto, conviene crear dos usuarios. 
(por defecto el u: admin, p:admin debería valer para hacer las pruebas)

Para crear un usuario que tenga permiso de administracion, se hace desde la consola (createsuperuser)
Para crear un usuario normal, desplegamos la barra lateral, le damos a Login y dentro de la ventana de Login le damos a 
Sign up. Nos pedirá nombre, una contraseña robusta y un correo electrónico.

######## FUNCIONALIDAD ######## 

Si accedemos a la página principal, vemos que tenemos un desplegable que permite navegar por las distintas secciones.
Nos logueamos en LOGIN:
*Si somos administradores:(tendremos que tener los permisos de administrador y habernos logueado, sino no podemos acceder)
	 tendremos una seccion llamada ADMINISTRAR que nos permite añadir una nueva prenda. En esta seccion podemos introducir nuevas
	 prendas con sus respectivas imágenes.

*Tanto si somos usuarios normales como administradores: 
	-Si pinchamos en una seccion vemos que cargarán los productos que se han elegido. 
	-Si nos metemos dentro de un producto, podremos ver el stock que tiene de cada talla y si lo queremos se puede añadir al carrito.
